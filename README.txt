SilverStripe LoginHistory Module
========================================

Maintainer Contacts
-------------------
*  Martijn van Nieuwenhoven (<info@axyrmedia.nl>)

Requirements
------------
* SilverStripe 2.4
* maxmindgeoip

Usage Overview
--------------

Simple DataObject to track Memberlogin locations
The login history is visible in the Member details tab.

Note that the Location data is stored serialized in one field, and can't be edditted at the moment.
